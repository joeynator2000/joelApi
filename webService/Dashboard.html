How to use the webapp:


Insert a country name into the input field above, and if the dataset includes this country, the gni, military spending, and suicide rates of this country will be graphed. The first two graphs are generated with JSON data, while the second two graphs are created using XML data! Try typing in Albania for example.


Important note: Countries with spaces in their names need to be spalled with under scores instead of spaces! E.g. russian federation needs to be spelled russian_federation!
